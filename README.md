# Saavira — Static Site (GitHub Pages)

Brand: **Saavira** — *Your Everyday Style*

This repo contains the production-ready static site for Saavira. To publish:

1. Create a public repo named `saavira-site` under GitHub user **saavira05**.
2. Upload all files from this folder to the repository root.
3. In **Settings → Pages**, set **Source** to `Deploy from a branch`, then choose `main` and `/ (root)`.
4. Your site will be live at: `https://saavira05.github.io/saavira-site/`.

### Custom domain
If you later gain DNS access, add a `CNAME` file containing your domain (e.g., `shop.saavira.com`) and create the appropriate DNS records at your registrar.
